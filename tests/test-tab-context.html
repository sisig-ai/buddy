Buddy Tab Context Test Page

Testing Tab Context

This page tests if Buddy correctly identifies and reads content from the current tab.

When you ask "What is this page?" or "What is this repo?", Buddy should:

  1. Receive the correct tab ID from the content script
  2. Use the read_page_content tool
  3. Read THIS page's content (not the sidebar's)
  4. Respond with information about THIS test page

Debug Information

Open the browser console to see debug messages:

  • Tab ID being used for content reading
  • Page content being sent to the API
  • Tool calls being made

Test Questions to Ask Buddy

  • "What is this page?"
  • "What is this page about?"
  • "Summarize this content"
  • "What debug information is mentioned here?"
